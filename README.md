# Chat Llama

一个现代化的AI聊天应用，提供类似ChatGPT的用户体验，支持多会话管理、实时响应和美观的蓝白主题界面。

## 功能特性

- 🤖 基于大型语言模型的智能对话
- 💬 多会话创建与管理
- 🔄 历史对话查看与继续
- 👤 用户注册与登录系统
- 🎨 清新蓝白主题界面
- 📱 响应式设计，同时支持PC和移动设备

## 技术栈

### 前端
- React 
- CSS3
- React Router

### 后端
- Golang
- Gin框架
- gRPC (与LLM服务通信)
- JWT认证
- SQLite/MySQL数据库
